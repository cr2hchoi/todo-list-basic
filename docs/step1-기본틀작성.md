# 🔍 Step 1: 기본 틀 및 레이아웃 작성 분석

프로젝트의 기초 뼈대가 되는 `index.html`의 주요 구성 요소 및 역할 정리.

### 1. 외부 라이브러리 및 스타일 설정 (`<head>`)

- **Bootstrap 5 (CDN)**: `unpkg.com`을 통해 외부 스타일시트 로드. 별도 CSS 작성 없이 깔끔한 UI(Card, Container 등) 활용 가능하게 설정
- **Custom CSS**: 프로젝트 전용 스타일 정의함 -`.body`: 문서의 default 값 설정  
  -`.title`: 앱 제목 중앙 정렬(text-align) 및 굵게 강조(font-weight:bold)
  - `.todo-done`: 완료된 항목 스타일 정의, 완료된 항목에 취소선(`line-through`) 적용
  - `.pointer`: 클릭 가능한 요소임을 알리기 위해 커서 모양을 변경한다. pointer로 지정하면 검지손가락 모양(👆🏻)이 나온다. -`.container`:반응형 영역을 지정한다.

### 2. 레이아웃 구조 (`<body>`)

- **`<div id="app">`**: Vue.js 인스턴스가 제어할 루트(Root) 엘리먼트. 모든 동적 데이터 바인딩의 기준점이 됨.
- **Card 컴포넌트**: 부트스트랩 클래스로 구역 시각적 분리 완료.
  - **상단**: 앱 제목(`::Todolist App`) 배치.
  - **본문**: 향후 할 일 목록과 입력창이 동적으로 생성될 공간 확보함.

### 3. 스크립트 엔진 (`<script>`)

- **Vue.js Core**: 최신 버전 라이브러리 로드. 자바스크립트 객체와 HTML 템플릿 연결 준비 완료.
- **Logic Space**: 향후 4단계에서 정의할 `addTodo`, `deleteTodo` 등 핵심 메서드가 작성될 영역!

### 🃏 Bootstrap Card 클래스 상세 분석

Bootstrap에서 제공하는 컨테이너 컴포넌트로, 콘텐츠를 박스 형태로 깔끔하게 구조화함.

- **`.card`**: 전체적인 카드 컨테이너 생성. 기본 테두리와 둥근 모서리 적용됨.
- **`.card-body`**: 카드 내부의 실제 콘텐츠 영역. 사방에 적절한 여백(Padding)을 자동으로 부여함.
- **`.bg-light`**: 카드의 배경색을 연한 회색으로 설정하여 시각적 구분감을 줌.
- **`.card-default`**: 부트스트랩 기본 카드 스타일 적용함.
- **`.card-borderless`**: (사용자 정의) 카드의 기본 테두리를 제거하여 더 심플한 UI 구현 시 사용함.
