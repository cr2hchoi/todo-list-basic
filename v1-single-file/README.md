# 📝 Todo List Basic Project

> HTML과 기초 JavaScript를 활용하여 할 일의 추가, 삭제, 완료 상태를 관리하는 심플한 투두 리스트 프로젝트연습

---

## 🎯 프로젝트 목표

- **직관적인 할 일 관리**: 사용자가 입력한 할 일을 실시간으로 목록에 추가하고 관리
- **상태 제어 시스템**: 완료된 할 일을 시각적으로 구분하고, 필요 없는 항목을 안전하게 삭제하는 기능 구현
- **Vanilla 환경 실습**: 외부 컴포넌트나 프레임워크 없이, 단일 파일 내에서 HTML/CSS/JS가 유기적으로 작동하는 구조로 연습

---

## 🛠️ 기술 스택 및 환경

- **Language**: `HTML5`, `CSS3`, `JavaScript (ES6+)`
- **UI Framework**: `Bootstrap 5 (via CDN)`
- **Development Style**:
  - 유지보수와 연습을 위해 외부 컴포넌트 방식이 아닌, **단일 코드 내에서 스타일과 스크립트를 모두 작성**하는 모놀리식 구조를 지향

---

## 🏗️ 개발 단계 (Roadmap)

### 1. 기본 인프라 구축

- [ ] Bootstrap 5 CDN 연동 및 기본 레이아웃 구성
- [ ] 사용자 정의 CSS 스타일 가이드 적용
- [ ] 프로젝트 기본 테마(Default Style) 설정

### 2. 정적 UI 디자인 및 하드코딩

- [ ] Bootstrap5 클래스를 활용한 컴포넌트 디자인
- [ ] 가상 데이터를 활용한 목록 렌더링 테스트
- [ ] JavaScript 배열 순회(`for`, `forEach`)를 통한 동적 데이터 바인딩 준비

### 3. 데이터 아키텍처 정의

- **`todo`**: 사용자 입력을 실시간으로 반영하는 데이터
- **`todolist`**: 할 일 객체들을 담는 메인 저장소(Array)
  - `id`: `Timestamp` 방식을 활용한 고유 키(Key) 생성
  - `content`: 할 일의 텍스트 내용
  - `completed`: 완료 상태를 제어하는 Boolean 값 (`true` / `false`)

### 4. 핵심 메서드 구현

- `addTodo`: 새로운 할 일을 목록에 추가하고 입력창 초기화
- `deleteTodo`: 선택한 고유 ID를 가진 항목을 목록에서 제거
- `toggleCompleted`: 할 일의 완료 상태를 반전시키고 UI 업데이트

### 5. 동적 템플릿 바인딩

- **데이터 바인딩**: 데이터의 변화가 UI에 실시간으로 투영되는 구조 작성
- **조건부 렌더링**: 완료 여부에 따른 취소선 처리 등 상태별 분기 처리
- **반복 렌더링**: 배열 데이터를 기반으로 리스트 아이템 자동 생성
